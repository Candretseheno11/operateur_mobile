# Operateur
# Concept
Une application server-side (CodeIgniter) permettant de proposer des régimes alimentaires adaptés aux utilisateurs selon leurs objectifs et leurs caractéristiques personnelles (sexe, taille, poids, activité, etc.).

Ce README décrit en détail l'architecture du projet, les étapes d'installation, la configuration locale, les commandes utiles, la structure des dossiers, la gestion de la base de données, l'exécution des tests et les bonnes pratiques de contribution.# operateur_mobile
